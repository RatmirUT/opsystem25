Teenuse otstarve:
BITS tegeleb taustal toimuvate failiedastustega, kasutades vaba võrguühendust. Seda kasutavad Windows Update, Microsoft Store’i rakendused, Microsoft Defender ning teised süsteemikomponendid, mis vajavad katkestuskindlat allalaadimist ja üleslaadimist.

Kas teenuse võib välja lülitada?
BITS-i saab ajutiselt välja lülitada, kuid see ei ole soovitatav. Selle väljalülitamine põhjustab Windows Update’i, Defenderi viirusetõrje uuenduste ja Microsoft Store’i rakenduste probleemse töötamise. Seega on kõige parem jätta teenus seisundisse Manual või Automatic, eriti kui süsteem peab saama uuendusi.
